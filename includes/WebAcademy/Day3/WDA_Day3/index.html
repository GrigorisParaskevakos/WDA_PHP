JavaScript

Η JavaScript (JS) είναι μια διερμηνευμένη (interpreted) γλώσσα σεναρίων (scripting) με συναρτήσεις πρώτης τάξεως (first-class functions).

Χρησιμοποιήθηκε αρχικά στους περιηγητές (browsers) για να προσφαίρει πρόσθετες δυνατότητες στις ιστοσελίδες, αλλά έχει πλέον επεκταθεί τόσο στην πλευρά του εξυπηρετητή (web-server) όσο και σε τρίτες πλατφόρμες.

Η JavaScript είναι βασισμένη σε πρωτότυπα (prototype-based), δυναμική (dynamic) και πολυπαραδειγματική (multi-paradigm) υποστηρίζοντας αντικειμενοστρεφή (object-oriented), προστακτικό (imperative) και δηλωτικό (declarative) στυλ προγραμματισμού.

Για να επωφεληθούμε από τις δυνατότητες που μας παρέχει η JavaScript γράφουμε μια σειρά από εντολές (κώδικα) ακολουθώντας ορισμένους κανόνες.

Τι κώδικα γράφουμε;

Ο κώδικας (και τα προγράμματα) της JavaScript είναι μια σειρά από εντολές που (διερμηνεύονται και) εκτελούνται μια προς μια με τη σειρά που έχουν γραφτεί.

Οι εντολές στην JavaScript ονομάζονται δηλώσεις (statements), διαχωρίζονται με το χαρακτήρα ";" και αποτελούνται από τιμές, τελεστές (operators), εκφράσεις (expressions), λέξεις κλειδιά (keywords) και σχόλια (comments).

Οι τιμές διαχωρίζονται σε σταθερές και μεταβλητές.

Μεταβλητές

Στη JavaScript, όπως και στις υπόλοιπες γλώσσες προγραμματισμού, αναπαριστούμε τις μεταβαλλόμενες τιμές στον αλγόριθμό μας με μεταβλητές.

Οι μεταβλητές έχουν όνομα και μπορούν να κρατήσουν αποθηκευμένη μια τιμή κατά τη διάρκεια της εκτέλεσης των εντολών του προγράμματός μας.

Τα ονόματα των μεταβλητών στη JavaScript πρέπει να είναι μοναδικά και να ακολουθούν τους εξής κανόνες:

  • Τα ονόματα των μεταβλητών πρέπει να ξεκινάνε από γράμμα (με εξαίρεση τους χαρακτήρες "$" και "_").
  • Τα ονόματα των μεταβλητών μπορούν να περιέχουν γράμματα, ψηφία και τους χαρακτήρες "$" και "_".
  • Τα ονόματα των μεταβλητών δεν πρέπει να αποτελούν λέξεις κλειδιά της JavaScript.

Στα ονόματα των μεταβλητών της JavaScript τα πεζά γράμματα διαφέρουν από τα κεφαλαία (case-sensitive).

Είναι καλή πρακτική οι μεταβλητές να δηλώνονται πριν χρησιμοποιηθούν σε κάποια έκφραση (expression).

Η δήλωση μιας μεταβλητής μπορεί να πραγματοποιηθεί με τη λέξη-κλειδί var ακολουθούμενη από το όνομα της μεταβλητής που θέλουμε χρησιμοποιήσουμε. Κατά τη δήλωση μιας μεταβλητής μπορεί να πραγματοποιηθεί και αρχικοποίησή της με μια τιμή.

Η τιμή κάθε μεταβλητής ανήκει σε έναν προκαθορισμένο τύπο δεδομένων.

Έλεγχος ροής

Κατά την επίλυση ενός προβλήματος μέσω ενός προγράμματος πολύ συχνά δημιουργείται η ανάγκη για εκτέλεση εντολών υπό ορισμένες συνθήκες. Η δυνατότητα ελέγχου της ροής ενός προγράμματος στηρίζεται στις δομές if...else και switch.

Δομή if...else


if (συνθήκη1) {

} else if (συνθήκη2) {

} else if (συνθήκη3) {

} else {

}

Παράδειγμα δομής if...else


var z = 5;
var a = 10;
var b = 20;
var c = 30;
var d = 40;
if (a >= b) {
    z += b;
} else if (a >= c) {
    z += c;
} else if (a >= d) {
    z += d;
} else {
    z += 50;
}

Δομή switch


switch(μεταβλητή) {
    case τιμή1:

        break;
    case τιμή2:

        break;
    default:

}

Παράδειγμα δομής switch


var a = 2;
var z = 0;
switch(a) {
    case 1:
        z += a;
        break;
    case 2:
        z += a * 2;
        break;
    default:
        z += 100;

}

Επανάληψη

Κατά την επίλυση ενός προβλήματος μέσω ενός προγράμματος πολύ συχνά δημιουργείται η ανάγκη για επαναληπτική εκτέλεση εντολών υπό ορισμένες συνθήκες. Η δυνατότητα επανάληψης εντολών στηρίζεται στις δομές for, while και do...while.

Δομή for


for (αρχικοποίηση; συνθήκη; τελευταία_εντολή) {

}

Παράδειγμα δομής for


var j = 0;
for (var i = 0; i < 10; i++) {
    j += i * 2;
}

Δομή while


while (συνθήκη) {

}

Παράδειγμα δομής while


var j = 0;
var i = 0;
while (i < 10) {
    j += i * 2;
    i += 1;
}

Δομή do...while


do {

} while (συνθήκη);

Παράδειγμα δομής do...while


var j = 0;
var i = 0;
do {
    j += i * 2;
    i += 1;
} while (i < 10);

Στη JavaScript μπορούμε να διακόψουμε την εκτέλεση του επαναληπτικού βρόχου (loop) με τη λέξη-κλειδί break και να προχωρήσουμε στην εκτέλεση της επόμενης επανάληψης με τη λέξη-κλειδί continue.

Συναρτήσεις

Οι συναρτήσεις είναι ένα σύνολο εντολών ομαδοποιημένων σε ένα μπλοκ κώδικα. Για να μπορούμε να χρησιμοποιήσουμε πολλαπλές φορές το ίδιο μπλοκ κώδικα δίνουμε σε αυτό ένα όνομα.

Για να καλέσουμε (να εκτελέσουμε τον κώδικα σε) μια συνάρτηση χρησιμοποιούμε το όνομά της ακολουθούμενο από παρενθέσεις στις οποίες εισάγουμε τα ορίσματα (παραμέτρους) της συνάρτησης.

Στην JavaScript οι συναρτήσεις μπορούν να εκχωρηθούν σε μεταβλητές και ως εκ τούτου μπορούν να περάσουν ως ορίσματα σε άλλες συναρτήσεις και να επιστραφούν ως αποτελέσματα κλήσης συναρτήσεων.

Η ονομασία των συναρτήσεων ακολουθεί τους ίδιους κανόνες με την ονομασία των μεταβλητών.

Η εκτέλεση των εντολών μιας συνάρτησης μπορεί να σταματήσει με τη λέξη-κλειδί return ακολουθούμενη προαιρετικά από κάποια τιμή που θέλουμε να επιστρέψουμε στο σημείο όπου έγινε η κλήση της συνάρτησης.

Παράδειγμα δήλωσης συνάρτησης:

function myFunction(a, b) {
    return a + b;
}

Παράδειγμα κλήσης συνάρτησης:
myFunction(5, 10);

Εμβέλεια μεταβλητών

Η εμβέλεια των μεταβλητών αφορά την ορατότητά τους στον κώδικα.

Οι μεταβλητές που δηλώνονται εντός των συναρτήσεων είναι τοπικές (local), μπορούν να χρησιμοποιηθούν μόνο εντός των συναρτήσεων που δηλώθηκαν και δεν είναι ορισμένες (undefined) έξω από το μπλοκ της συνάρτησης που δηλώθηκαν.
Οι μεταβλητές που δηλώνονται έξω από συναρτήσεις έχουν καθολική εμβέλεια (global) και είναι ορατές από όλα τα σημεία του κώδικα.

Συμβάντα (events)

Κάθε αλληλεπίδραση του χρήστη με μια ιστοσελίδα είτε είναι η μετακίνηση του ποντικιού ή το πάτημα ενός πλήκτρου στο πληκτρολόγιο αποτελεί ένα συμβάν (event). Μέσω της JavaScript δίνεται η δυνατότητα στον προγραμματιστή να εκτελέσει κώδικα σε προκαθορισμένα συμβάντα.

Ακολουθούν μερικά ενδεικτικά συμβάντα:

Συμβάν Περιγραφή
onclick Ενεργοποιείται με κάθε πάτημα πλήκτρου στο ποντίκι και με κάθε πάτημα στις οθόνες των φορητών συσκευών.
onchange Ενεργοποιείται με την αλλαγή κατάστασης ενός στοιχείου συνήθως σε φόρμες.
onblur Ενεργοποιείται με κάθε φορά που ένα στοιχείο χάνει την εστίαση (focus) του χρήστη.
onsubmit Ενεργοποιείται κατά την αποστολή μιας φόρμας
onload Ενεργοποιείται κατά το φόρτωμα ενός στοιχείου και συνήθως χρησιμοποιείται στο body.
onscroll Ενεργοποιείται κατά την κύλιση (scrolling) ενός στοιχείου και συνήθως χρησιμοποιείται στο body.

Για να χειριστούμε ένα συμβάν χρησιμοποιούμε τη μέθοδο document.addEventListener().

Cookies

Τα cookies είναι δεδομένα που αποθηκεύονται στον περιηγητή τόσο για την διατήρηση πληροφοριών που αφορούν τις επιλογές και προτιμήσεις ενός χρήστη σε ένα site όσο και για την ανάκτηση της συνεδρίας (session) ενός χρήστη σε κάθε νέο αίτημα (request) που πραγματοποιείται.

Τα cookies που έχουν οριστεί για ένα site αποστέλλονται σε κάθε νέο αίτημα που πραγματοποιείται προς το site.

Η πρόσβαση στα cookies γίνεται μέσω της ιδιότητας document.cookie.

Εγκυρότητα δεδομένων

Ένα από τα μεγαλύτερα προβλήματα που συναντούν οι περισσότερες διαδικτυακές και μη εφαρμογές είναι ο έλεγχος εγκυρότητας των δεδομένων. Για να είναι μια εφαρμογή φιλική προς το χρήστη πρέπει όχι μόνο να εντοπίζει τα λανθασμένα δεδομένα του χρήστη αλλά και να τον καθοδηγεί στην εισαγωγή των κατάλληλων για την εκάστοτε εφαρμογή δεδομένων.

Για την εγκυρότητα και τον έλεγχο των δεδομένων μπορεί να βοηθήσει τόσο η HTML μέσω των ιδιοτήτων που έχουν τα πεδία εισαγωγής δεδομένων, όσο και η JavaScript με τον έλεγχο των δεδομένων κατά την εισαγωγή τους και πριν την αποστολή τους, καθώς και με την εμφάνιση κατάλληλων μηνυμάτων προς το χρήστη που καθοδηγούν τη διαδικασία εισαγωγής πληροφοριών σε μια φόρμα.

jQuery - AJAX

Η κύρια ίσως λειτουργία που μειώνει τόσο το φόρτο εργασίας για έναν εξυπηρετητή (server) όσο και για τον περιηγητή (browser) βελτιώνοντας παράλληλα την εμπειρία χρήσης (UX) είναι η επικοινωνία με αιτήματα AJAX.

Τα αιτήματα AJAX πραγματοποιούνται στο παρασκήνιο όσο ο χρήστης αλληλεπιδρά με μια σελίδα ή τη διαβάζει.

Η αποστολή δεδομένων σε έναν εξυπηρετητή με χρήση AJAX παραμετροποιείται εύκολα με τη μέθοδο $.ajax() της βιβλιοθήκης jQuery. Εναλλακτικά μπορεί να χρησιμοποιηθεί το αντικείμενο XMLHttpRequest χωρίς να απαιτείται η χρήση κάποιας βιβλιοθήκης κώδικα.

Παράδειγμα χρήσης της μεθόδου $.ajax()


$.ajax({
    method: "POST",
    url: "some.php",
    data: { name: "John", location: "Boston" }
})
.done(function( msg ) {
    alert( "Data Saved: " + msg );
});
